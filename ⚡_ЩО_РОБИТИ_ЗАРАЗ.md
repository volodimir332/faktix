## 🔥 ТІЛЬКИ 2 КРОКИ:

### КРОК 1: Налаштувати Firebase Console (5 хв)

1. **Відкрийте:** https://console.firebase.google.com/
2. **Оберіть проект:** faktix-8d2cc
3. **Перейдіть:** Authentication → Sign-in method
4. **Натисніть на:** Google (в списку провайдерів)
5. **Увімкніть:** перемикач "Enable"
6. **Виберіть:** Support email (ваш email)
7. **Натисніть:** Save
8. **Прокрутіть вниз до:** Authorized domains
9. **Натисніть:** Add domain
10. **Введіть:** `faktix.cz`
11. **Натисніть:** Add

✅ **Готово!**

   ↓
   Enable [✓] ← Поставити галочку
   ↓
   Support email: [ваш email] ← Вибрати
   ↓
   Save ← Натиснути
   ```

3. **Додати домен:**
   ```
   Authorized domains (внизу сторінки)
   ↓
   Add domain ← Натиснути
   ↓
   faktix.cz ← Ввести
   ↓
   Add ← Натиснути
   ```
